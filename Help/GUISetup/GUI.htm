This is the screen that allows you to create, or modify, an SDR name. The screens will be broadly similar in appearance on all platforms, however the choices available will vary. The availablility of the Set buttons depends on the type of SDR that is specified in the SDR file. These buttons are:

  • User Audio Setup
  • SDR Audio Setup
  • Tuning Hardware Setup
  • External Program Setup

If you fail to set some of the values that are needed then an error will appear when you press the Create button.
