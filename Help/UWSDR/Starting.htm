The UWSDR GUI may be started in a number of ways. Depending on the operating system and the options selected in GUI Setup not all of them may be available to you.

  • Clicking on icons created by GUI Setup in either a program menu or from the Desktop.


  • Typing the command in at a command line shell, in the form

    UWSDR <SDR Name>

    where <SDR Name> is the name given in GUI Setup. This option does not work correctly with Mac OS X.


  • Typing the command in at a command line shell, in the form

    UWSDR

    i.e. with no SDR Name argument. In this case the Config Chooser box is shown and a list of existing configurations are given to choose from.

    Clicking either the Cancel button or the close window icon will abort the UWSDR GUI.
