This sets the IF bandwidth of the SDR (really the DSP). It can be one of:

  • 20.0 kHz
  • 15.0 kHz
  • 10.0 kHz
  • 6.0 kHz
  • 4.0 kHz
  • 2.6 kHz
  • 2.1 kHz
  • 1.0 kHz
  • 500 Hz
  • 250 Hz
  • 100 Hz
  • 50 Hz
  • 25 Hz
  • Auto

When in the Auto mode the bandwidth will be dependent on the mode in use, which is set in the Modes Preferences section.
