The Receive Menu allows for the monitoring of the input signal levels at various stages along the receive chain. The taps are at:

  • I Input level, is the raw in-phase signal from the hardware.
  • Q Input level, is the raw quadrature signal from the hardware.
  • Strength, is the signal level after filtering, but before the AGC.
  • Avg Strength, is a smoothed version of the Strength above.
