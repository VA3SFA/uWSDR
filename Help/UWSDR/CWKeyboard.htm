The CW Keyboard entry on the Menu brings up a new window where a number of pre-set messages and a real-time keyboard may be chosen from. Real-time keying is possible by using an external key input via the serial port which can be set up via GUISetup. Pressing the external key while sending a message from the CW Keyboard will abort it.

When using one of the pre-existing messages, the transmitter will go off when the message is complete or the Abort button is pressed. When using the real-time typing space at the bottom, the transmitter only goes off when the End button is pressed.
