This complex screen sets the defaults for the filter size, tuning rate and AGC/Deviation. The settings for SSB are the values that will be used for both USB and LSB.

The tuning speed references the values set on the Step Size preferences screen. The setting for FM Narrow is taken from the value used for FM Wide.

The filter setting from here is the value used when the Auto setting is set from the Filter setting on the main screen, see that help entry for the available values.

The AGC/Deviation sets the deviation value for FM, which can be one of:

  • 6 kHz
  • 5 kHz
  • 3 kHz
  • 2.5 kHz
  • 2 kHz

For the other modes, the AGC value can be set here. It can be one of:

  • Fast
  • Medium
  • Slow
  • None

There is no seperate setting for CW Narrow as it takes its AGC value from the CW Wide setting. The Deviation settings are not operational when the SDR is receive only.
