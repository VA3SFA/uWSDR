The Voice Keyer is used to transmit a voice file instead of using the microphone while in a voice mode. The file used must be a WAV file, and using the Browse button, a file may be selected, as well as a list of previouslmy used files which is also available. The file may be transmitted once by selecting the Single repetition, or looped continuously until either the Abort button is pressed. The operation of the buttons at the bottom of the screen is the same as that for the CW Keyboard.
